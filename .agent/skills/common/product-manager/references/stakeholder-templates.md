# Stakeholder Communication Templates

## EOD Email Template (Daily — for direct manager)

```
Subject: EOD Update — [Date] — [Project Name]

🟢/🟡/🔴 Status: [On Track / At Risk / Off Track]

✅ Done today
• [Item 1] — [one line impact]
• [Item 2] — [one line impact]

🔄 In Progress
• [Item] — [% complete] — on track for [date]

🔴 Blockers
• [Blocker] — blocked since [date] — needs: [action] from [owner]
• (None)

📅 Tomorrow
• [Priority 1]
• [Priority 2]
• [Priority 3]
```

## Delay Notification Template (Async — for stakeholders)

```
Subject: ⚠️ [Feature Name] — Delivery Update

Hi [Name],

I'm writing to flag that [Feature Name] will be delayed from [original date] to [new date].

**Root cause**: [1-2 sentences — technical complexity / dependency / scope change]

**Impact**: [What this affects — other features, commitments, etc.]

**Mitigation**: [What we're doing to minimize impact]

**Next update**: [Date]

Happy to discuss. [Your name]
```
