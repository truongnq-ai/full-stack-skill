# Overview

tech-stack-skill tập trung vào tư duy hệ thống và quy trình thực thi mã nguồn theo best practices cộng đồng.

## Mục tiêu kép
1) Onboarding dev nhanh và nhất quán
2) Điều hướng AI agent với decision tree + conventions rõ ràng

## Nguyên tắc
- Ưu tiên chuẩn cộng đồng + thực dụng
- Có checklist rõ ràng để không bỏ sót bước quan trọng
- Luôn nêu anti-patterns để tránh sai lầm phổ biến
