# full-stack-skill

Bộ skill chuyên sâu theo tech stack để:
- ✅ Onboarding dev (coding convention + setup + structure)
- ✅ Auto guidance cho AI agent (decision tree + best practices)

## Mục tiêu
- Chuẩn hóa kiến trúc + coding style theo best practices cộng đồng
- Giúp AI agent ra quyết định nhất quán theo stack

## Cấu trúc
- language/
- framework/
- infra/
- architecture/
- templates/

## Cài đặt
Repo có CLI (Node ESM) tương tự `global-skill`.
Kết quả cài đặt sẽ nằm tại: `.agent/skills/`.

> Xem `docs/CLI.md` để dùng CLI.

## Format chuẩn của mỗi skill
- Overview & Context
- Coding conventions (style, lint, naming, folder layout)
- Project structure
- Common patterns
- Testing
- Security & pitfalls
- CI/CD checklist
- AI agent playbook (steps/decisions)
- Do/Don’t (anti-patterns)

## Versioning
Date-based `YYYY.MM.DD`
