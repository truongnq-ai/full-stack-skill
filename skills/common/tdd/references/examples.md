# Examples — TDD

## Example 1 — Red First

**Input**
"Implement feature then add test"

**Output**
"Write failing test first, then implement."

**Why**
- Ensures test validity.

---

## Example 2 — Minimal Green

**Input**
"Over-engineer before test passes"

**Output**
"Implement smallest change to pass."

**Why**
- Avoids scope creep.
