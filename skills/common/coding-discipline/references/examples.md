# Examples — Coding Discipline (Refined)

## Example 1 — Ambiguity Detected

**Input**
"Optimize performance" (no target)

**Output**
"Ask for scope/metrics; propose options; wait for confirmation."

**Why**
- Prevents unbounded refactors.
