# Examples — iOS Design System (Refined)

## Example 1 — Typography

**Input**
```swift
label.font = .systemFont(ofSize: 16)
```

**Output**
```swift
label.font = DS.Typography.body
```

**Why**
- Consistent styles.

---

## Example 2 — Colors

**Input**
```swift
view.backgroundColor = .blue
```

**Output**
```swift
view.backgroundColor = DS.Colors.primary
```

**Why**
- Centralized palette.
